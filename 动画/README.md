# 动态效果(含过渡和动画)

过渡和动画是两个不同的概念

## 过渡

起始状态、目标状态明确，指定之后元素会从起始状态缓缓变为目标状态

时间确定

## 动画

起始状态、目标状态不明确，通常不用指定

时间通常也不确定
